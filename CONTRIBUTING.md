# Contributing

Thank you for considering contributing to NodeByte Hosting, Dokploy Module for WHMCS! We welcome contributions from the community.

## How to Contribute

1. **Fork the repository**: Click the "Fork" button at the top right of this page.
2. **Clone your fork**: 
    ```sh
    git clone https://github.com/your-username/module-dokploy-whmcs.git
3. **Create a new branch**: 
    ```sh
    git checkout -b feature-branch
    ```
4. **Make your changes**: Implement your feature or fix.
5. **Commit your changes**: 
    ```sh
    git commit -m 'Add some feature/fix'
    ```
6. **Push to the branch**: 
    ```sh
    git push origin feature-branch
    ```
7. **Open a pull request**: Go to the repository on GitHub and click "New pull request".

## Reporting Issues

If you find a bug or have a feature request, please create an issue on GitHub.

## Special Instructions

- **Contribution/Commit Branches**: All new contributions/commits can be pushed directly to the development branch or a new branch created by you during the pull request process.
